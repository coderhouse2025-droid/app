## 📊 Descripción

**GastoApp Pro** es una aplicación web progresiva (PWA) diseñada para gestionar ingresos y gastos de forma intuitiva, moderna y eficiente.

Permite visualizar tu situación financiera en tiempo real, analizar tus gastos por categoría y exportar reportes profesionales en PDF, todo directamente desde el navegador y sin backend.

---

## ✨ Features

* 📈 Dashboard financiero en tiempo real
* 💵 Registro de ingresos y gastos
* 🧠 Balance automático y porcentaje utilizado
* 📊 Gráficos dinámicos por categoría (Chart.js)
* 📅 Filtro por mes y año
* 📄 Exportación a PDF profesional
* 💾 Persistencia con LocalStorage
* 📱 Instalación como app (PWA)
* 🎨 UI moderna con TailwindCSS + animaciones

## 🛠️ Instalación

```bash
git clone https://github.com/coderhouse2025-droid/App.git
cd App
```

Abrir:

```bash
index.html
```

Opcional (recomendado para PWA):

```bash
npm install -g live-server
live-server
```

## 📊 Funcionalidades

### 📌 Gestión de movimientos

* Alta de ingresos y gastos
* Clasificación por categorías
* Eliminación de registros

### 📊 Dashboard

* Balance disponible
* Total ingresos vs gastos
* Porcentaje de uso del presupuesto

### 📈 Visualización

* Gráfico tipo doughnut
* Indicadores de consumo (normal / warning / danger)

### 📄 Exportación

* PDF con resumen financiero
* Tabla detallada de movimientos
* Diseño profesional listo para compartir
